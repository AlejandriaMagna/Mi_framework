Acerca de mí

Este framework fue diseñado para el curso: Diseño y desarrollo web + IA, de la Universidad Católica de Temuco.

Objetivos

  • -Para demostrar que podemos tener nuestro propio Bootstrap
  • -Tener un diseño/platilla hecha con anterioridad para ahorrar tiempo
  • -Personalizar nuestra página web con un estilo original

Aprendizajes

Mejorar habilidad con elementos CSS y desrrollar la creatividad, teniendo presente la accesibilidad y usabilidad.

Desafíos

Hacer una estructura HTML semántica, ya que esta información fue extraída de W3School, pero no cuenta con etiquetas section o aside, sólo div..
